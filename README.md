# Mini Wiki Dark 📚

[👉 웹에서 바로 사용해보기 (라이브 데모)](https://choe050919.github.io/MiniWiki/)

**Mini Wiki Dark**는 브라우저의 로컬 스토리지를 활용하여 서버 없이 동작하는 초경량 개인 위키입니다. 마크다운 문법, 위키 스타일 문서 간 연결, 버전 관리, 다크 모드 등을 지원합니다.

> **참고**: 위 링크를 통해 접속하여 생성하는 문서는 사용자 브라우저에 안전하게 저장되며, 어떠한 서버로도 전송되지 않습니다.

## ✨ 주요 기능

*   **서버 불필요**: 별도의 백엔드 설치 없이 'index.html' 파일을 실행하는 것만으로 바로 사용할 수 있습니다.
*   **데이터 프라이버시**: 모든 데이터는 브라우저 내부(로컬 스토리지)에만 저장됩니다.
*   **마크다운 편집**: 마크다운 문법을 지원하는 간단한 텍스트 편집기와 실시간 미리보기를 제공합니다.
*   **코드 블록 강화**: 코드 블록에 대한 구문 강조('highlight.js' 통합)를 제공합니다.
*   **위키 링크**: '[[문서이름]]' 또는 '[[문서이름|표시텍스트]]' 형식으로 문서 간 자유로운 이동 및 생성이 가능합니다.
*   **목차 및 백링크**: 현재 문서의 구조를 보여주는 **목차(TOC)** 및 이 문서를 참조하는 **백링크**가 사이드바에 자동으로 생성됩니다.
*   **히스토리 관리**: 문서의 수정 이력을 저장하고 특정 버전으로 복원할 수 있습니다.
*   **사이드바 기능**:
    *   왼쪽: 전체 문서 목록 (가나다/최근 순 정렬, 필터링), 고정된 문서 관리 (드래그 앤 드롭 정렬).
    *   오른쪽: 현재 문서 목차, 백링크.
*   **데이터 백업**: JSON 형식으로 데이터를 내보내기/가져오기를 지원합니다.
*   **테마 지원**: 다크 모드와 라이트 모드 전환 지원 (기본 테마는 다크 모드입니다).
*   **파일 드래그 앤 드롭**: `.md` 또는 `.txt` 파일을 브라우저 창으로 끌어다 놓아 새 문서로 즉시 가져올 수 있습니다.
*   **문서 삭제**: UI를 통해 더 이상 필요 없는 문서를 간편하게 삭제할 수 있습니다.
*   **외부 링크 구분**: 본문 내 외부 링크 옆에는 🔗 아이콘이 자동으로 표시되어 내부 링크와 쉽게 구분됩니다.

## 📖 사용 방법

### 1. 문서 탐색 및 생성
*   상단 **입력 바**에 문서 제목을 입력하고 'Enter' 키를 누릅니다.
*   문서가 존재하면 해당 문서로 이동하고, 그렇지 않으면 새 문서가 생성됩니다.
*   'All'을 입력하여 전체 문서 목록을 볼 수 있습니다.

### 2. 문서 편집
편집 모드에 진입하면 좌측에 마크다운 편집기가, 우측에 실시간 미리보기가 나타납니다.
*   **내부 링크**: '[[링크 문서 이름]]' 또는 '[[링크 문서 이름|보여줄 텍스트]]' (마크다운 모드)
*   **외부 링크**: '[구글](https://google.com)' (마크다운 모드)

### 3. 명령 (입력 바)
*   '문서 이름': 해당 문서로 이동/생성
*   'All': 모든 문서 목록 보기
*   ':history [문서 이름]': 현재 또는 특정 문서의 히스토리 보기

### 4. 키보드 단축키
| 단축키       | 동작                        |
| :----------- | :-------------------------- |
| **Ctrl + E** | 편집 모드로 전환            |
| **Ctrl + S** | 저장 (편집 모드에서)        |
| **Ctrl + H** | 현재 문서 히스토리 보기     |
| **Esc**      | 취소 / 보기 모드로 돌아가기 |

## 💾 백업 및 복구
데이터는 브라우저에 저장되므로, 브라우저 캐시를 지우면 데이터가 삭제될 수 있습니다.
상단 메뉴의 **내보내기(Export)** 버튼을 클릭하여 '.json' 파일로 데이터를 정기적으로 백업해주세요.

## 🛠 기술 스택
